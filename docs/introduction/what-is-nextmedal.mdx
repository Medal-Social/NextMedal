## Introduction

NextMedal is a cutting-edge, open-source template that combines the power of Next.js 14, Sanity CMS, and modern web technologies to help you build scalable social platforms. Whether you're creating a community platform, a social network, or a content-sharing site, NextMedal provides the foundation you need.

## Key Features

<Steps>
  <Step title="Modern Stack">
    Built with Next.js 14, React 18, TypeScript, and Tailwind CSS for a robust and maintainable codebase.
  </Step>
  
  <Step title="Content Management">
    Integrated Sanity CMS with custom schemas and real-time updates for flexible content management.
  </Step>
  
  <Step title="Authentication">
    Secure authentication with NextAuth.js, supporting multiple providers and custom workflows.
  </Step>
  
  <Step title="UI Components">
    Beautiful, accessible components built with Radix UI and styled with Tailwind CSS.
  </Step>
  
  <Step title="Performance">
    Optimized for speed with static generation, incremental static regeneration, and edge functions.
  </Step>
  
  <Step title="Developer Experience">
    Enhanced DX with TypeScript, ESLint, Prettier, and Husky for code quality.
  </Step>
</Steps>
