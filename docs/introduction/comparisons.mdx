# Comparisons

This page compares NextMedal with other popular frameworks and solutions to help you make an informed decision.

## NextMedal vs Next.js

| Feature | NextMedal | Next.js |
| ------- | --------- | ------- |
| Framework | Built on Next.js | Base framework |
| CMS Integration | Sanity built-in | Requires manual setup |
| UI Components | Pre-built components | None included |
| Documentation | Fumadocs built-in | Requires manual setup |
| Theme Support | Built-in light/dark mode | Requires manual setup |
| SEO | Optimized by default | Requires manual setup |

**When to choose NextMedal over Next.js:**
- You want a complete solution with CMS integration
- You need pre-built UI components
- You want a faster development experience
- You prefer structured content with Sanity

## NextMedal vs Other Headless CMS Solutions

| Feature | NextMedal | Contentful | Strapi | WordPress |
| ------- | --------- | ---------- | ------ | --------- |
| CMS Type | Sanity (headless) | Headless | Headless | Traditional/Headless |
| Pricing | Open source | Freemium | Open source | Open source |
| Real-time collaboration | ✅ | ✅ | ❌ | ❌ |
| Structured content | ✅ | ✅ | ✅ | Limited |
| Custom schemas | ✅ | Limited | ✅ | Limited |
| Development experience | Excellent | Good | Good | Varies |
| Frontend integration | Seamless | Manual | Manual | Manual |

**When to choose NextMedal over other CMS solutions:**
- You want a tightly integrated Next.js + Sanity experience
- You prefer JavaScript/TypeScript for your entire stack
- You need flexible content modeling
- You want a modern developer experience

## NextMedal vs All-in-one Website Builders

| Feature | NextMedal | Webflow | Wix | Squarespace |
| ------- | --------- | ------- | --- | ----------- |
| Customization | Full code access | Limited code access | Limited code access | Limited code access |
| Performance | High | Medium | Medium | Medium |
| Developer control | Full | Limited | Limited | Limited |
| Complexity | Medium | Low | Low | Low |
| Hosting flexibility | Any provider | Webflow only | Wix only | Squarespace only |

**When to choose NextMedal over website builders:**
- You need complete control over your code
- You want maximum performance
- You need custom functionality
- You prefer a developer-centric workflow
