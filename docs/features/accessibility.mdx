# Accessibility

NextMedal is built with accessibility in mind, making your site usable for everyone and improving your SEO at the same time.

- **How it works:**  
  The template uses semantic HTML, ARIA attributes, and supports keyboard navigation throughout all components.

- **Why it matters:**  
  Accessible sites reach a wider audience and are favored by search engines.

[Learn more about web accessibility](https://www.w3.org/WAI/fundamentals/accessibility-intro/)
