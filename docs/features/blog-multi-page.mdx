# Blog & Multi-Page Support

NextMedal is designed to handle blogs and complex multi-page sites right out of the box. You can easily create, organize, and optimize content for both single and multi-page experiences.

- **How it works:**  
  The template provides schemas and routing for blog posts, categories, and any number of custom pages, all managed in Sanity Studio.

- **Why it matters:**  
  Well-structured content helps search engines understand your site and improves discoverability for users.
