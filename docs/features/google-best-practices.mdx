# Built for Google Best Practices

All components in NextMedal are designed with [Google's SEO best practices](https://developers.google.com/search/docs/fundamentals/seo-starter-guide) in mind. This ensures your site is search-friendly and follows the latest guidelines for discoverability and ranking.

- **How it works:**  
  The template uses semantic HTML, proper metadata, accessibility features, and performance optimizations recommended by Google.

- **Why it matters:**  
  Following best practices helps your site rank higher and ensures a great experience for both users and search engines.
